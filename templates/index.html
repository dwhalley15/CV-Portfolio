About This Page

This web page is a simple, full stack, web application. Hosted on replit.com, it uses Python Flask as a web framework. When the app is run, a Python function returns the custom HTML file, which displays the page. The page design is described in a separate custom CSS file stored on the server and linked via HTML. The page design follows the ethical design principal PARC and utilises a monochromatic colour scheme. The interactive buttons trigger JavaScript functions (JavaScript file is also stored on the server and linked via HTML) to manipulate the document object model and dynamically display the relevant sections on the page. The displayed data shown when these functions are run is stored on the server in JSON files. The JavaScript functions use HTTP GET requests to trigger server-side Python functions, which read the JSON files and return the data requested. For full access to the code used (as well as other projects) please follow the link to my replit account in the contact details section.
